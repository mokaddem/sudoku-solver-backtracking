The script reads a sudoku supplied in a file.

Each line is an array where digits are separated by commas. Emtpy cell are represented by the '*' character.

```
Example:

    [5,3,*,*,7,*,*,*,*]
    [6,*,*,1,9,5,*,*,*]
    [*,9,8,*,*,*,*,6,*]
    [8,*,*,*,6,*,*,*,3]
    [4,*,*,8,*,3,*,*,1]
    [7,*,*,*,2,*,*,*,6]
    [*,6,*,*,*,*,2,8,*]
    [*,*,*,4,1,9,*,*,5]
    [*,*,*,*,8,*,*,7,9]
```
